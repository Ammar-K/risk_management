# المبادئ التوجيهية


| إدارة المخاطر                                        | كيف يتم تطبيقها؟                                                                                          | لماذا هي مهمة؟                                                        |
| ---------------------------------------------------- | --------------------------------------------------------------------------------------------------------- | --------------------------------------------------------------------- |
| 1. خلق وحماية القيمة                                 | - مدمجة في إطار الحوكمة                                                                                   | - تساهم في تحقيق الأهداف                                              |
|                                                      | - تعتبر جزءًا من ثقافة المنظمة                                                                            | - تساعد على تحسين الأداء                                              |
|                                                      |                                                                                                           | - تحمي مصالح المجتمع                                                  |
| 2. جزء لا يتجزأ من عملية التخطيط والإدارة في الوكالة | - مدمجة في التخطيط الاستراتيجي وتخطيط الأعمال                                                             | - تتجنب التكرار                                                       |
|                                                      | - تُعلم التفويضات                                                                                         | - توجه تحديد الأولويات                                                |
|                                                      | - جزء من عمليات إدارة التغيير                                                                             | - توضح المسؤوليات                                                     |
| 3. جزء من عملية اتخاذ القرار                         | - مدمجة في عمليات الموافقة                                                                                | - تساعد صناع القرار على اتخاذ قرارات مستنيرة                          |
|                                                      | - مدرجة بشكل صريح في المشاريع وتصميم الأنظمة والتغيرات وتخصيص الموارد                                     | - تساعد في تحديد الأولويات                                            |
|                                                      | - جزء من جميع الاتفاقيات التعاقدية                                                                        | - تميز بين الخيارات البديلة للإجراءات                                 |
|                                                      | - جزء من ترتيبات التوظيف والتوظيف                                                                         |                                                                       |
| 4. تعالج عدم اليقين بشكل صريح                        | - تُستخدم لتطوير أوصاف لمعايير تقييم المخاطر (مثل الاحتمالية والنتيجة)                                    | - تحدد بشكل صريح عدم اليقين في السياقات الداخلية والخارجية للوكالة    |
|                                                      | - مرتبطة بتقييم الأهداف                                                                                   | - تعزز رؤية مشتركة للمخاطر وشهية المخاطر                              |
|                                                      |                                                                                                           | - تحدد نقاط الضعف                                                     |
| 5. منظمة، مهيكلة وفي الوقت المناسب                   | - مدمجة في تصميم جميع الأنظمة بدلاً من أن تكون عملية مستقلة                                               | - تساهم في نهج متسق وفعال                                             |
|                                                      | - تُطبق باستمرار من خلال توجيه واضح                                                                       | - تسهل المقارنة بين النتائج والقياس المرجعي                           |
|                                                      | - تُقاس وتُبلغ                                                                                            | - تعزز الفهم المتسق                                                   |
| 6. تستند إلى أفضل المعلومات المتاحة                  | - يتوفر المشورة والدعم لإدارة المخاطر                                                                     | - يحتاج أصحاب المصلحة إلى بيانات دقيقة وموثوقة لإدارة المخاطر         |
|                                                      | - تحدد المتطلبات الوظيفية لأنظمة إدارة المخاطر                                                            | - تدعم الشهادة بالمخاطر                                               |
|                                                      | - تُستخدم لتحديد عدم اليقين بدقة وضمان أن العلاجات ذات صلة                                                | - تقيم فعالية الضوابط                                                 |
|                                                      |                                                                                                           | - تطور مراقبة المخاطر والإبلاغ عنها                                   |
|                                                      |                                                                                                           | - أنظمة معلومات إدارة المخاطر مناسبة للغرض                            |
| 7. مصممة بشكل مخصص                                   | - إطار المخاطر مصمم ومشغل ليتناسب مع سياق وقدرات الوكالة                                                  | - يتماشى مع السياق الداخلي والخارجي للوكالة وملف المخاطر              |
|                                                      | - يتماشى مع ثقافة الوكالة                                                                                 | - تُخصص موارد كافية                                                   |
|                                                      |                                                                                                           | - الامتثال للالتزامات القانونية                                       |
| 8. يأخذ العوامل البشرية والثقافية في الاعتبار        | - إطار المخاطر يأخذ في الاعتبار كيفية تفاعل الأشخاص والثقافات مع وظائفه وكيفية مراقبة ثقافة وسلوك المخاطر | - يتماشى مع قدرات ونوايا أصحاب المصلحة مع أهداف المنظمة               |
|                                                      |                                                                                                           | - يضمن التوافق بين الثقافة والسلوك                                    |
| 9. شفاف وشامل                                        | - يحدد نطاق وطريقة مراقبة المخاطر والإبلاغ عنها لأصحاب المصلحة                                            | - يحدد العناصر المطلوبة في معايير المخاطر                             |
|                                                      | - يحدد دور أصحاب المصلحة في عملية إدارة المخاطر                                                           | - يعزز رؤية المخاطر بين جميع مستويات الوكالة                          |
|                                                      |                                                                                                           | - يسهل المشاركة المناسبة وفي الوقت المناسب لأصحاب المصلحة             |
|                                                      |                                                                                                           | - يضمن أن استراتيجية إدارة المخاطر تظل ذات صلة ومحدثة                 |
| 10. ديناميكي وتكراري ويستجيب للتغيير                 | - مدمج في استراتيجيات إدارة التغيير                                                                       | - يضمن أن الاستراتيجية تأخذ في الاعتبار المخاطر الناشئة               |
|                                                      | - مدمج في الخطط الاستراتيجية وخطط الأعمال                                                                 | - يضمن أن إطار إدارة المخاطر يستجيب للتغيرات في السياق                |
|                                                      |                                                                                                           |                                                                       |
| 11. يسهل التحسين المستمر                             | - نظام إدارة المخاطر مدمج في أنظمة التحسين المستمر                                                        | - تُستخدم شهادة المخاطر ونتائج التدقيق الداخلي لإعلام التحسين المستمر |
|                                                      | - يتم السعي إلى ملاحظات أصحاب المصلحة للتأثير على التطوير المستمر لإطار المخاطر                           | - يحسن نضج المخاطر في الوكالة                                         |
|                                                      |                                                                                                           | - يعالج توقعات أصحاب المصلحة لحماية مصالح المجتمع                     |
|                                                      |                                                                                                           | - يساعد المنظمة على الوفاء بالالتزامات                                |
